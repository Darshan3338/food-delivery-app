# Food Delivery APP
## Instalation Process
    - npm create vite@latest food-delivery-app
    - Framework: React
    - Variant: JavaScript
    - cd food-delivery-app
    - npm install
    - npm install react-router-dom bootstrap axios
    - add Bootstrap in min.jsx --- import 'bootstrap/dist/css/bootstrap.min.css';
    - npm run dev
